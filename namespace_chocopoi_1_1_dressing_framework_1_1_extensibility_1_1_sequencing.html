enum Chocopoi.DressingFramework.Extensibility.Sequencing.BuildStage

Build stage

Enumerator
Pre 

Generic pre-stage. The earliest stage to execute hooks. You are preferred to categorize your hook instead of using this generic stage.

Preparation 

Preparation stage. Scanning, cloning etc. are done in this stage.

Generation 

Generation stage. Component generation, cloning are done in this stage.

Transpose 

Transpose stage. Mapping, animation etc. general stuff are done in this stage.

Optimization 

Optimization stage

Post 

Generic post-stage. The latest stage to execute hooks. You are preferred to categorize your hook instead of using this generic stage.
